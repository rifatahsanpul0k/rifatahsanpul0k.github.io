# Rifat Ahsan Pulok - Personal Portfolio

A premium, Apple-inspired personal portfolio website built for an undergrad CS student.

## Features
- **Apple Aesthetics**: Clean, minimalist design with glassmorphism and bento grids.
- **Bento Grid Layout**: Modular layout for showcasing skills, projects, and experience.
- **Scroll Animations**: Smooth entrance animations using Intersection Observer.
- **Responsive**: Fully responsive design for mobile, tablet, and desktop.
- **Glassmorphism**: Sophisticated blur effects and subtle borders.

## Tech Stack
- **Core**: HTML5, Vanilla JavaScript
- **Styling**: CSS3 (Vanilla)
- **Bundler**: Vite
- **Icons**: Lucide Icons

## Getting Started
1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`

## Design System
The design follows the "Apple Senior UI Designer" persona, prioritizing:
- Precise typography (Inter/SF Pro)
- High contrast and generous white space
- Subtle micro-interactions
- Consistent 24px border radius
